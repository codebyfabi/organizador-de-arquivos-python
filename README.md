# Organizador de Arquivos Automático 📁

Um script em Python desenvolvido para automatizar a organização da pasta de Downloads, movendo arquivos para pastas específicas baseadas em suas extensões.

## Funcionalidades
- **Organização Inteligente:** Move arquivos (.docx, .pdf, .exe, .jpg, .png, .html) para pastas organizadas automaticamente.
- **Criação Dinâmica:** Cria as pastas de destino caso elas ainda não existam no seu computador.
- **Segurança:** Utiliza caminhos dinâmicos, garantindo que o script funcione em qualquer máquina sem expor dados pessoais.

## Como utilizar
1. Certifique-se de ter o Python instalado.
2. Clone este repositório ou baixe o arquivo `Organizador_documentos.py`.
3. Execute o script. Ele irá organizar automaticamente sua pasta de Downloads!

## Tecnologias utilizadas
- Python (bibliotecas `os` e `shutil`)
